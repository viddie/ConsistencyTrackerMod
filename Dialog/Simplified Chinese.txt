# Newlines automatically create a Page Break, unless there is an {n}
 command on the previous line
# Commands: Anything inside of curly braces {...} is a command and should not be translated.

# Inline Text Commands:
# {~}wavy text{/~}
# {!}impact text{/!}
# {>> x}changes speed at which characters are displayed{>>}
# {# 000000}this text is black{#} (uses HEX color values)
# {+MENU_BEGIN} inserts the dialog from the MENU_BEGIN value (in English, "CLIMB")
# {n}
 creates a newline, without a page break
# {0.5} creates a 0.5 second pause

    在CCT的初期，只有原始统计数据会输出到文本文件。外部应用程序必须自行计算所有有意义的统计数据以利用这些原始数据。{n}{n}
    实时数据功能增加了由模组端计算的统计数据，并允许用户决定如何对它们进行格式化
    撰写本FAQ时，已有大约80种不同的统计数据和来自当前游戏状态的信息可以通过实时数据输出。

MODOPTION_CCT_FAQ_LIVE_DATA_WHAT_IS_LIVE_DATA_FORMAT_Q= 什么是实时数据格式？
MODOPTION_CCT_FAQ_LIVE_DATA_WHAT_IS_LIVE_DATA_FORMAT_A=
    “格式”指示实时数据如何输出信息。每个格式都有一个名称和包含某些占位符的任意文本。{n}
    占位符的格式如下：{category:name}。实时数据会将所需的统计数据插入占位符的位置。{n}{n}
    示例：格式{n}
    “当前：{room:name} | 房间：({room:roomNumberInChapter}/{chapter:roomCount})”{n}
    在游玩Farewell（第九章）时可能会输出{n}
    “当前：DT-7 | 房间：(59/93)”。

MODOPTION_CCT_FAQ_LIVE_DATA_DONT_SEE_XY_IN_MY_LIST_OF_FORMAT_Q= 我没有在格式列表中看到"xy"，它在哪里？
MODOPTION_CCT_FAQ_LIVE_DATA_DONT_SEE_XY_IN_MY_LIST_OF_FORMAT_A=
    当更新中添加新的默认格式时，它们不会自动添加到你的格式列表中，因为这可能会覆盖你手动做出的更改。
    要获取最新的默认格式，你可以通过以下按钮将'format.txt'文件重置为默认：{n}
    模组选项 -> !!数据清除!! -> 重置 'format.txt' 文件{n}
    但请注意，这会重置你创建的所有自定义格式，因此请先备份它们！

MODOPTION_CCT_FAQ_LIVE_DATA_HOW_CAN_I_MAKE_FORMAT_Q= 我该如何创建自己的格式？
MODOPTION_CCT_FAQ_LIVE_DATA_HOW_CAN_I_MAKE_FORMAT_A=
    有两种方法可以创建自己的格式：{n}{n}
    格式编辑器{n}
    1. 通过“模组选项 -> 实时数据设置 -> 在浏览器中启动格式编辑器”启动格式编辑器{n}
    2. 在格式编辑器中，点击左下角的“新格式”按钮{n}
    3. 给你的格式起一个名字。{n}
    4. 在格式文本部分输入你想要的内容。你可以在右侧面板中看到所有可用的占位符。
将鼠标悬停在占位符上可以查看它的解释，点击占位符可以将其插入到格式文本中。如果你在地图中，你可以在底部的框中预览格式的显示方式。{n}
    5. 点击底部的“创建新格式”按钮以保存你的格式{n}{n}
    手动文件编辑{n}
    1. 通过“模组选项 -> 实时数据设置 -> 打开格式文本文件”在文本编辑器中打开'format.txt'文件{n}
    2. 在文件底部为你的自定义格式添加新行{n}
    3. 按照'名称;格式'的模式编写你的格式。要查看所有可用的占位符并获得实时预览，我仍然推荐使用格式编辑工具。{n}
    4. 保存文件{n}
    5. 通过“模组选项 -> 实时数据设置 -> 重新加载格式文件”重新加载格式文件{n}{n}
    新添加的格式现在应该可以在游戏内的展示窗口中选择了！
